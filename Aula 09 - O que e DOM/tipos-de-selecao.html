Aprendendo a usar o DOM em JavaScript.

Há várias formas de acessar elementos do DOM para fazer alterações.

Acessando a arvore DOM até agora do site temos: 'window. ==> document. ==> body. ==> p'

Porém, neste caso, acessariamos todas as tags 'p' dentro de body. Para evitarmos isso, temos os seletores feitos das seguintes maneiras:

  • Por Marca
  • Por Id
  • Por Class
  • Por Seletor

Selecionando elementos por Marca (tagName)

Alternado este texto através do seletor por marca com o comando:

window.document.getElementsByTagName('p')[4].style.color = 'blue'

Como são vários p's dentro do documento html, é necessário dizer entre [] qual o número do p que você quer selecionar

Selecionando elementos por Id

Alternado este texto através do seletor por ID

forma diferente de fazer a alteração é jogar o dado em uma variável e alterar a variavel


var seletor_id = window.document.getElementById('exemplo-id')

seletor_id.style.color = 'yellow'

seletor_id.style.border = '1px solid black'

Selecionando elementos por Class

Alternado esse texto com a mesma class de outros textos através do seletor por Class

Texto com a mesma class do anterior

Assim como a por Marca, neste caso também é necessário dizer qual Class deve ser acessada

Omitir o window também funciona

var seletor_class = document.getElementsByClassName('exemplo-class')[0]

seletor_class.style.fontSize = '16px'

seletor_class.style.border = '1px solid black'

Selecionando elementos por Seletor

Alternado esse texto por seletor

Esta propriedade usa os seletores do CSS para acessar as propriedades (como '#' ou ' . ' para class ou Id)

var seletor = document.querySelector('.seletor')

seletor.style.border = '1px solid black'
